# 📂 Dataset

## Dataset Information

The dataset used for this project was provided during my Data Analytics Internship at Codec Technologies.

Due to data privacy and ownership restrictions, the original dataset is not included in this public repository.

## Project Workflow

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Customer Churn Prediction

This repository demonstrates the complete machine learning workflow using Python, Pandas, Scikit-learn, Random Forest, Logistic Regression, and XGBoost.
